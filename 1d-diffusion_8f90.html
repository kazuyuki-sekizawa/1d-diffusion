program diffusion_1d

Solves a 1D diffusion equation with the FTCS method.

Author
Kazuyuki Sekizawa
Date
2022/05/20
See also
https://nuclphystitech.wordpress.com/undergrad/comptphys/step-3/

This program solves a one-dimensional diffusion equation,

using a forward difference for time derivative and a central difference for spatial derivative.

Parameters
iSpatial index
ip1Spatial index for i+1
im1Spatial index for i-1
iterTime index
Niter_maxThe maximum number of time steps to be computed
dxGrid spacing
dtTime step
kappaDiffusion coefficient
f(1:Nx)An array of the function to be evolved in time
f_new(1:Nx)A buffer for the array f
char_iterStores 'iter' as a 4-digit character, e.g., 0001, 0002, ...
Precondition
Modify the input file, "input.txt".
Specify the initial condition for f(x)

Compile

$ gfortran 1d-diffusion.f90

Execution

$ ./a.out
